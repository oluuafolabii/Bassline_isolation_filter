### **Project Title: Design and Implementation of a Low-Pass FIR Filter for Bassline Isolation in Audio Signals using the Windowing Method**

**Objective:** The objective of this project is to design and implement a low-pass FIR filter using the windowing method. The project aims to isolate low-frequency components (such as a bassline or kick drum) from a complex audio signal, like an EDM or Alternative Rock track. It will compare the filter's frequency response to that of an ideal low-pass filter  and investigate its applications in audio signal processing and equalization.

**Methodology:**

1. Understand the theory and principles of low-pass filtering, the windowing method, and discrete-time convolution.

2. Select a suitable, multi-layered audio signal (e.g., a .wav file of an EDM or Alternative Rock song) that requires low-pass filtering and evaluate its frequency domain characteristics using MATLAB.

3. Design the FIR low-pass filter using the windowing method with a selected filter length and a cutoff frequency suitable for bass isolation (e.g., 200 Hz to 250 Hz).

4. Implement the filter and apply it to the audio signal using MATLAB's digital signal processing toolsets.

5. Evaluate the filter's frequency response, magnitude response, and phase response using MATLAB.

6. Compare the filter's frequency response to that of an ideal low-pass filter and evaluate its performance.

7. Compare the time-domain waveforms and frequency spectrums of the original audio signal against the filtered audio signal to verify successful isolation.

**Deliverables:**

1. MATLAB code for designing the low-pass filter using the windowing method and processing the .wav audio file.

2. A report detailing the theory and principles of low-pass filtering and the windowing method, the properties of the analyzed audio signal, and an analysis of the filter's frequency response, magnitude response, and phase response.

3. A comparison of the filter's performance to that of an ideal low-pass filter.

4. Exported audio files demonstrating the original track and the isolated bassline.

**Grading:** The project will be graded on the following components:

1. MATLAB code: 20%

2. Report: 40%

3. Presentation: 40%

Note: The report and presentation will be weighted equally, each accounting for 40% of the final grade. The MATLAB code will account for 20% of the final grade.
